# Untitled Chronicle RPG

A deep persistent-world RPG about discovering power words, combining them into
impossible capabilities, and leaving material consequences in a generated
world that outlives every body the player inhabits. A mature Chronicle grows
from strange discovery into Holdings, autonomous Agents, factional history, and
the world-changing possibility of a Palimpsest.

Goal 2, Slice 3, and Goal 4 are complete. Combat, Explore, and Build now open
onto the same Chronicle: the authored Codex supports deliberate Study choices,
`Found` establishes the singular persistent Home and its physical Return
Route, and `Smash` can leave the Riven Cairn durably shattered. Slice 4C and
Goal 4 passed player UAT on 2026-07-20. Slice 5, A Word Multiplies, is the
latest completed gate: its shared authored Expression resolution, `Fly[Bell]`
journey, strict save v5, retained automated gate, and player UAT passed on
2026-07-20. The player later rejected collectible `Verb[Noun]` composition as
the future fun axis. The accepted successor direction is Verbs plus linked
Modifiers acting on contextual world Targets. Its isolated combat-grammar
pressure test is active on a throwaway branch; all successor production work
remains separately gated and unauthorized. Goal 6 remains later. The
engine-independent Chronicle simulation owns every rule and persistent value,
while Godot renders its state and translates player input.

Start with [the vision](docs/VISION.md), [the glossary](CONTEXT.md), and [the
codemap](docs/CODEMAP.md). The [roadmap](docs/ROADMAP.md) owns sequencing and
the [active handoff contract](docs/HANDOFF.md) keeps current work inside its
gate. Build, test, editor, and launch instructions are in [the development
guide](docs/DEVELOPMENT.md). The project is deliberately untitled; do not
invent a public name before the game earns one.

The [Modifier Grammar Course Correction](docs/MODIFIER-GRAMMAR-DIRECTION.md)
records the successor direction and its deliberately unresolved decisions.
The active [Combat Grammar Pressure Test](docs/COMBAT-GRAMMAR-PRESSURE-TEST.md)
runs with `& .\prototype-combat.ps1` and never enters production projects or
saves.

From the repository root, build and play with the normal save using
`.\play.ps1`. Use `.\play.ps1 -Fresh` for a new isolated Chronicle or
`.\play.ps1 -Profile <name>` to create or continue a named one.
