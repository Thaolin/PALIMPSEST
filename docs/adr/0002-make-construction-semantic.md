# Make construction teach the Universe

PALIMPSEST will not center conventional crafting/building/combat or present
Knowledge as an unlockable cheat menu. Its expressive core is Semantic
Construction: players group material Entities into an Arrangement, offer it as
an Exemplar for a taught Archetype, let the recognized result become a Semantic
Construction, bind learned meaning through a Patch Anchor, and live with the
compositional consequences. Survival supplies pressure, building supplies
physical syntax, and combat tests the resulting rules; local anchored changes
precede broad Reach so increasing power continues to create play.

## Consequences

- Placement, spatial relationships, Archetype teaching, and Patch Anchors are
  first-class progression systems rather than decorative base building.
- Scarred Zones use the same Patch, Knowledge, and Provenance machinery as
  player constructions instead of bespoke anomaly scripts. Their bounded
  spatial binding must be designed explicitly before implementation.
- Fever-dream outcomes must remain consistent enough to investigate. Random
  incoherence is not a substitute for semantic composition.
