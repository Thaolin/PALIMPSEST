# Untitled Chronicle RPG — Project Instructions

Follow [AGENTS.md](AGENTS.md) as the canonical project guide. In particular:

- Build a C#-only Godot 4 .NET game with an engine-independent Chronicle core.
- Keep the game pausable and fixed-tick; it is not a reflex-action RPG.
- Treat Codex persistence, Agent autonomy, material Holdings, and causal
  Chronicle Records as core promises.
- Use `CONTEXT.md` for the project’s language and `docs/` for vision,
  architecture, roadmap, ADRs, references, and handoff state.
- Do not revive the deleted PALIMPSEST implementation or its gameplay model.
